# Política de Privacidade — CNPJ/CPF Remover Pontuação

**Última atualização:** abril de 2026

## Coleta de dados
Esta extensão **não coleta, não armazena e não transmite** nenhum dado do usuário.

## Funcionamento
Todo o processamento é feito localmente no próprio navegador do usuário.
Nenhuma informação digitada ou copiada é enviada a servidores externos.

## Permissões utilizadas
- **contextMenus:** exibir opção no menu de botão direito
- **clipboardWrite:** copiar o resultado para a área de transferência
- **scripting / activeTab:** exibir confirmação visual na aba ativa
